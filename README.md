# 💳 Fraud Detection Model

## 🚀 Overview

This project focuses on detecting fraudulent financial transactions using machine learning techniques. The model is designed to handle class imbalance and maximize fraud detection recall.

## 🎯 Objective

To build a high-performance fraud detection system that minimizes false negatives while maintaining precision.

## 🧠 Features

* Data preprocessing and feature engineering
* Handling imbalanced data using SMOTE
* LightGBM model implementation
* Achieved ~95% fraud recall
* Reduced false positives significantly

## 🛠️ Tech Stack

* Python
* LightGBM
* Scikit-learn
* Pandas, NumPy
* Jupyter Notebook
